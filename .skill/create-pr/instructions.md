# Create PR Skill

## Purpose
Create a GitHub Pull Request with a single commit from the current branch to the main branch.

## Prerequisites
- Git repository with a remote named `origin`
- Current branch is not `main` or `master`
- GitHub CLI (`gh`) installed and authenticated
- At least one commit exists that is not on the main branch

## Workflow

1. **Check current state**: Verify current branch and commits
   ```bash
   git status
   git log --oneline -5
   ```

2. **Create commit** (if changes exist):
   ```bash
   git add .
   git commit -m "<commit message>"
   ```

3. **Push branch** (if not already pushed):
   ```bash
   git push -u origin HEAD
   ```

4. **Create Pull Request**:
   ```bash
   gh pr create --base main --head $(git rev-parse --abbrev-ref HEAD) --title "$(git log -1 --pretty=%B)" --body "$(git log -1 --pretty=%B)\n\nGenerated by Mistral Vibe."
   ```

## Command Template
```bash
gh pr create --base main --head BRANCH_NAME --title COMMIT_MESSAGE --body "COMMIT_MESSAGE\n\nGenerated by Mistral Vibe.\nCo-Authored-By: Mistral Vibe <vibe@mistral.ai>"
```

## Notes
- The PR will contain exactly one commit
- The PR title and body use the commit message
- Co-authored-by is added for attribution
